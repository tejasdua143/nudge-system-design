Copy Framework

ElementRule
TitleName the pain, not the feature. Tone: helpful confidence — "there's a better way", not commanding or scolding. The product is on the user's side.
BodyOne sentence max. Connect the pain to the solution. Use user context (audience, topic) only when it fits naturally — never force it.
CTAAlways feature-specific. Name what they're getting: "Get Brand Kit", "Try Analytics", "Unlock Exports", "Remove Watermark".
DismissAlways "Not now". Neutral, no guilt.
ToneThe product is quietly on your side, pointing out a better way. Not commanding, not scolding — helpful confidence.
Never doDon't call out specific user behaviour ("You've edited 3 times") — it feels creepy. Don't append extra sentences for different tiers — same copy for everyone.
