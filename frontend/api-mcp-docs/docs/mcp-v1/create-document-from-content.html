MCP Integration: Connect Presentations.AI to AI Assistants

Presentations.AI exposes an MCP (Model Context Protocol) server that lets AI assistants create, transform, and edit presentations through natural language. Connect once, then just ask:

"Create a 10-slide pitch deck about AI in healthcare for executives"

"Turn these meeting notes into a 5-slide summary"

"Make slide 3 more concise and add a conclusion slide"

Prerequisites

  • A Presentations.AI account (sign up free)
  • Claude.ai / Claude Desktop: No further setup needed. OAuth handles login automatically
  • Claude Code / VS Code / Cursor: An API key from developers.presentations.ai (log in, go to Settings > API, and generate your key. It starts with pai_)

Overview

MCP server URL:

https://developers.presentations.ai/mcp

8 tools covering the full presentation workflow:

Tool What it does
create_presentation_topic "Make a deck about X" - AI generates all slides from a topic string
create_single_slide "Add one slide about Y" - AI generates a single slide on any subject
create_presentation_from_slides Build from structured slide-by-slide JSON with layout, audience, and tone control
create_document_from_content Exact content - pass a JSON slide array and get a presentation with no AI rewriting
create_presentation_from_content "Turn these notes into slides" - transform raw text with 4 preservation modes
create_presentation_from_file "Convert this PDF" - upload PDF, Word, or PPT and get a new presentation
update_document_content "Fix slide 3" - edit, add, or remove slides in an existing presentation by document ID
check_job_status Poll a running job - called automatically by Claude when generation takes time

Supports 6 export formats (PPTX, PPT, PDF, image, web preview, shareable link), multiple audience types, tones, and 4 content preservation modes.

Authentication

Two paths, depending on which surface you use:

OAuth 2.1 (Automatic)

For: Claude.ai, Claude Desktop, Claude Mobile

A login window opens automatically when you connect via the Connectors UI. Sign in with your Presentations.AI account and approve access. No API key needed.

API Key (Manual)

For: Claude Code, VS Code, Cursor

You pass your API key as a Bearer token in the configuration.

How to get your API key:
  1. Go to developers.presentations.ai and log in
  2. Navigate to Settings > API
  3. Click Register for API, then click Reveal to see your key
  4. Copy the key (starts with pai_) - store it securely

Claude.ai (Web)

  1. Go to Settings (gear icon) in Claude.ai
  2. Click Connectors
  3. Click Add custom connector
  4. Enter the MCP server URL: 
    https://developers.presentations.ai/mcp
  5. Click Connect - a login window will open
  6. Sign in with your Presentations.AI account and approve access
  7. Done - 8 tools are now available in your conversations

No API key needed. OAuth handles authentication automatically. The connection syncs to Claude Mobile as well.

Claude Desktop

  1. Open Claude Desktop and go to Settings
  2. Click Connectors
  3. Click Add custom connector
  4. Enter the MCP server URL: 
    https://developers.presentations.ai/mcp
  5. Click Connect - your browser will open for login
  6. Sign in and approve access
  7. Return to Claude Desktop - tools are ready

Important: Remote MCP servers must be added via the Connectors UI, not via claude_desktop_config.json. The config file is only for local stdio-based servers.

Claude Code (CLI)

Run this command in your terminal:

claude mcp add presentations-ai \
  --transport http \
  https://developers.presentations.ai/mcp \
  --header "Authorization: Bearer YOUR_API_KEY"

Replace YOUR_API_KEY with your Presentations.AI API key. Log in at developers.presentations.ai, then go to Settings > API > Register for API > Reveal to get your key.

Then ask Claude Code naturally:

Create a 5-slide presentation about renewable energy

VS Code (Copilot MCP)

Add to your .vscode/settings.json:

{
  "mcp.servers": {
    "presentations-ai": {
      "type": "http",
      "url": "https://developers.presentations.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}

Replace YOUR_API_KEY with your API key. Restart VS Code after saving.

Cursor

Add to your Cursor MCP configuration:

{
  "mcpServers": {
    "presentations-ai": {
      "url": "https://developers.presentations.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}

Replace YOUR_API_KEY with your API key.

Example Usage

Once connected on any surface, just describe what you need in natural language. Claude will pick the right tool automatically.

Create from topic

"Create a 10-slide pitch deck about AI in healthcare for executive leadership in PPTX format"

Transform content

"Turn these meeting notes into a 5-slide summary presentation: [paste notes]"

Edit existing

"Make slide 3 more concise and add a new conclusion slide"

Single slide

"Create a title slide for my quarterly business review"

Verify the Connection

After connecting, verify that all 8 tools are available:

  • Claude.ai / Desktop: Start a new conversation and ask "What Presentations.AI tools do you have?" - Claude should list all 8 tools.
  • Claude Code: Run claude mcp list and look for presentations-ai with 8 tools.
  • VS Code: Open the Command Palette and search for "MCP" to see connected servers.

Try a quick test: "Create a single slide about company vision" - you should get a presentation URL back within seconds.

Troubleshooting

Issue Cause Fix
Connect button does nothing (Claude.ai) Server URL incorrect or server unreachable Verify the URL is exactly https://developers.presentations.ai/mcp
OAuth login opens but redirects to dashboard Browser session conflict Try in an incognito/private window, or clear cookies for presentations.ai
Auth error (401) Invalid or expired API key Log in at developers.presentations.ai, go to Settings > API, and regenerate your key.
Tools not showing Connection not established Restart the application and reconnect. For Claude Code, run claude mcp list to verify.
Insufficient credits Account credits exhausted Log in at developers.presentations.ai to check your balance and add credits.

Additional Notes

  • Transport: The MCP server uses Streamable HTTP transport (MCP protocol 2025-06-18).
  • Export formats: All tools support ppt, pptx, pdf, image, render, and share formats.
  • Async jobs: Some presentations take longer to generate. Claude will automatically poll for completion using the check_job_status tool.
  • Security: API keys should never be committed to version control or shared publicly. Store them as environment variables.
  • Mobile: Connectors added on Claude.ai (web) are automatically available on Claude Mobile (iOS/Android). You cannot add new connectors directly on mobile.

For the full REST API documentation, see the API Reference. For support, contact support@presentations.ai.
