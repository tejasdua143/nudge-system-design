Account Signup

To start using our API you need to register with our platform. Following are the steps involved:

1. Register for an account

Go to the developer portal at developers.presentations.ai/docs/login

You can sign up using your LinkedIn/Google/Microsoft credentials or use your own email to sign up. However you will have to confirm your account when you use your own credentials.

Note: If you have already signed up on our platform then you can directly login

2. Request for API Key

Click on the Register for API button.

3. API Details

After signup, the api key is displayed. Click on the Reveal button to see the key. Use this key as Bearer Token in your API requests. On Sign-up 200 credits are given to you.

Click on the Reset button to generate a new API Key.
